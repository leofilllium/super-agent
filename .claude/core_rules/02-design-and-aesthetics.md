# UNIVERSAL DESIGN & ASSET STANDARDS
**UI/UX Aesthetic:**
1. **Primary Palette:** All applications must utilize a "forest green" (`#077632`) primary color palette for branding, primary buttons, and active states. 
2. **Interface Paradigm:** Default to highly professional, data-rich UI designs. Prioritize a "Palantir/Gotham" aesthetic—centralized dashboards, dense information hierarchies, dark mode optimization, and high-contrast terminal-like data tables.
3. **Component Structure:** Enforce strict centralization of UI components. Reusability is paramount.

**Asset Generation Protocol:**
Whenever a task requires the generation of visual assets (e.g., UI icons, 3D isometric vehicle renders, map overlays), you must append the following strict instruction to the generator:
**"BRUH IT IS NOT AN ICON/LOGO, JUST GENERATE AN IMAGE... MAKE A BACKGROUND COLOR WHITE. All generated app icons or assets must be presented as raw images on a solid white background rather than demonstrative or stylized mockups."**
